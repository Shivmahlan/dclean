# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in `dcleaner`, please **do not open a
public issue**. Instead, email the maintainer at **shivmahlan141@gmail.com**
with:

- A description of the vulnerability and its impact
- Steps to reproduce (or a proof of concept)
- Any suggested mitigation, if you have one

You can expect an acknowledgment within a few days. Once the issue is confirmed
and a fix is prepared, we will coordinate a release and credit you (unless you
prefer to remain anonymous).
